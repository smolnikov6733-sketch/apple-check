Установка Shortcut

Чтобы быстро открыть чек-лист из приложения «Команды»:

  1. Откройте Команды → нажмите + → Добавить действие → найдите «Открыть URL».
  2. Вставьте URL: https://smolnikov6733-sketch.github.io/apple-check/
  3. Сохраните ярлык и при желании добавьте на главный экран.
